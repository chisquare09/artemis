## Special Units

### A1.5 — Operational Vertical Slice ★
Currently the fully operational unit with:
- 7 activities (intro, vocabulary, reading, listening, speaking, writing, quiz)
- Deterministic exercises
- Listening activity
- Speaking roleplay
- Materials ingestion support
- AI tutor integration

### Certification Units
- **B1.11**: Certification Skills: B1 Foundation
- **B2.11**: Certification Preparation: B2

Both include academic_certification objectives for listening, reading, writing, speaking, and grammar exam skills.

### Capstone/Integration Units
- **A1.10**: A1 Survival Project
- **A2.10**: A2 Integration Project
- **B1.12**: B1 Integration Project
- **B2.12**: B2 Capstone Project

## Objective Types

Each unit uses flat objectives with type/content format:
- `communicative_goal`: Real-life communication abilities
- `grammar`: Structural language patterns
- `vocabulary`: Topic-based word groups
- `listening`: Listening comprehension skills
- `speaking`: Speaking production skills
- `reading`: Reading comprehension skills
- `writing`: Writing production skills
- `culture`: Cultural competence
- `academic_certification`: Exam and certification skills
